Composer and Magento

We now use Composer for dependency management. Composer enables us to manage the Magento components and their dependencies. Composer provides you with the following advantages: * Enables you to reuse third-party libraries without bundling them with source code * Component-based architecture with robust dependency management * Manages dependencies to reduce extension conflicts and compatibility issues * Versioned dependencies * Semantic versioning * Supports the PHP Framework Interoperability standard
